# Risk-Management
### Quantitative risk management notes and code (ongoing)

This repository contains my ongoing notes and implementations of quantitative risk management methods.

Many of the methods are reproductions of existing work, and in many cases include adaptations, modifications, and extensions. Each notebook documents its sources and relevant details where available.

This repository is intended as a personal reference for my own work and research, and will continue to expand over time as new ideas, techniques, and insights are added.
